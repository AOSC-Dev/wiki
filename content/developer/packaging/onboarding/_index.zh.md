# 前言

本文章旨在循序渐进地让热衷于 Linux 开发的读者深入安同 OS 的开发环境、开发工具和开发习惯。本文章相对于技术说明较为友好，并且尽量做到讲出细节，因此读者将无需在阅读本文期间查阅技术参考文档。

本文主要分为三部分：

1. 发行版打包基础：介绍软件编译、构建系统、发行版维护的基础事项
2. 安同 OS 打包入门：Ciel! 容器化打包环境、ABBS 软件包树、构建脚本编写及测试流程
3. 安同 OS 打包进阶：ABBS 树内的 Git 维护规范、主题制软件包更新机制及 BuildIt! 自动化构建设施

附录中将总结安同 OS 维护中可能会遇到的问题及解决流程，供贡献者参考。

请从右侧选择章节开始阅读吧！祝您的安同 OS 贡献旅程顺利。
