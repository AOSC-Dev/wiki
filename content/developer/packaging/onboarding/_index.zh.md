# 什么是 Linux？

严格意义上说，Linux 是一个开源的操作系统内核。Linux 最初由 Linus Torvalds 编写，经过三十余年的发展，现在由全球数万开发者参与维护。但是要组成我们日常所使用的操作系统，只靠内核往往不够：Linux 内核只负责操作系统的关键部分，如分配计算机资源、驱动设备、管理进程等。

现代操作系统为了提高安全性，会将操作系统分为内核空间和用户空间。我们平时使用的交互式提示符（Shell）、各种命令、图形界面，都是由用户空间的软件提供的。Linux 内核在完成初始化之后，就由用户空间的各个组件将系统引导至准备好使用的状态。

所以简单地说，这里的 “Linux” 指的是 Linux 发行版，是由 Linux 内核和用户空间组成的完整的操作系统。

# Linux 内核和发行版的关系

Linux 内核是开放源代码的，因此任何人都可以编译并分发 Linux 内核。由个人或组织基于 Linux 内核源代码发行的 Linux 内核通常被称为 Linux 内核的下游发行。 用户空间程序也是如此：软件的开发者（我们称之为上游）发布源码，然后个人或组织（“下游”）负责适配并编译，然后分发（Distribute）给用户。

完整的 Linux 操作系统由 Linux 内核和用户空间程序组成，所以个人或组织发行的 Linux 内核和用户空间程序的集合被称为是 Linux 发行版。因此，Linux 发行版看起来像是独立的操作系统。
