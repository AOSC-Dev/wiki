# 导言

近期，龙芯中科武汉公司给我社赞助了一台 LoongArch 构建服务器 (Relay 27863)，但构建暂存盘为机械硬盘，性能较差，故计划购买一块二手 Intel 750 SSD。

因为目前我们已购买此 SSD，本次众筹的目的是报销此费用。

# 预算计划

此块 800GB 容量的 Intel 750 SSD 价格为 699 人民币。

# 日程计划

目前已收到 SSD 并寄往龙芯中科武汉公司。
